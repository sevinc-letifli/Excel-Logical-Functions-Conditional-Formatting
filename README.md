# Excel-Logical-Functions-Conditional-Formatting
Excel Data Analysis project using Logical Functions and Conditional Formatting

## Project Overview
This project demonstrates the application of advanced Microsoft Excel techniques to process, analyze, and visually interpret data. The primary focus is on automating decision-making processes using logical functions and enhancing data readability through dynamic conditional formatting.

## Key Features & Tasks Completed:
- **Logical Functions (IF):** Implemented complex `IF` statements to automatically categorize data, calculate conditional outcomes, and assign specific statuses based on defined business rules.
- **Conditional Formatting:** Highlighted critical data points using dynamic rules:
  - *Pink:* Drinks category
  - *Green:* Foods category
  - *Yellow:* Sales > 10 and Quantity <= 3
  - *Grey:* Blank cells
- **Data Structuring:** Cleaned and organized the raw dataset to ensure high accuracy and logical flow for reporting.
- **Visual Insights:** Designed the layout to make large tables easy to read and interpret at a single glance for stakeholders.

## Tools Used:
- Microsoft Excel (Advanced Formulas, Logical Functions, Conditional Formatting)

# Excel Data Analysis: Logical Functions & Conditional Formatting

## Layihə Haqqında
Bu layihə məlumatların emalı, analizi və vizual şərhi üçün Microsoft Excel-in qabaqcıl texnikalarından istifadəni nümayiş etdirir. Əsas məqsəd məntiqi funksiyalar vasitəsilə qərar qəbul etmə proseslərini avtomatlaşdırmaq və dinamik şərti formatlaşdırma ilə məlumatların oxunurluğunu artırmaqdır.

## Görülən Əsas İşlər:
- **Məntiqi Funksiyalar (IF):** Müəyyən edilmiş biznes qaydalarına əsasən məlumatların avtomatik kateqoriyalara ayrılması, statusların təyin edilməsi və şərti nəticələrin hesablanması üçün `IF` funksiyası tətbiq olunub.
- **Şərti Formatlaşdırma (Conditional Formatting):** Məlumatları daha oxunaqlı etmək üçün xüsusi rəng qaydaları tətbiq olunub:
  - *Çəhrayı:* Drinks (İçkilər) kateqoriyası
  - *Yaşıl:* Foods (Qidalar) kateqoriyası
  - *Sarı:* Satış (Sales) > 10 və Say (Quantity) <= 3 olan sətirlər
  - *Boz:* Boş xanalar
- **Məlumatların Strukturlaşdırılması:** Xam məlumatlar təmizlənib və hesabatlılıq üçün məntiqi ardıcıllığa gətirilib.
- **Vizual Hesabatlılıq:** Böyük cədvəllərin bir baxışda rahat oxunması və analiz edilməsi üçün vizual struktur tərtib edilib.

## İstifadə Olunan Alətlər:
- Microsoft Excel (Advanced Formulas, Logical Functions, Conditional Formatting)
